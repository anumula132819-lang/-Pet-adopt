# 🐾 Pet Adoption Website

A modern and responsive Pet Adoption Website designed to connect loving families with pets in need of a home. This project focuses on creating a user-friendly platform with an attractive UI/UX that encourages users to adopt pets through engaging campaigns and easy navigation.

## Features
 *Home Page** with eye-catching hero section and mission statement
*  *Available Pets Section** displaying pets with images, details, and adopt buttons
* *Campaigns Section** to promote adoption drives and awareness programs
*  **Adoption Form** with input validation for user submissions
* *About Us Page** explaining the purpose and mission
*  *Contact Page** with sample contact details and form
*  Fully **Responsive Design** (mobile + desktop)
*  Modern UI with cards, hover effects, and smooth animations
  

##  Images
The website uses high-quality pet images sourced from free platforms like Unsplash, Pexels, or Google for visual appeal.

##  Technologies Used
* HTML5
* CSS3
* JavaScript

## Objective
The main goal of this project is to promote pet adoption and provide an easy-to-use platform where users can explore pets, participate in campaigns, and submit adoption requests.

##  How to Run
1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in any web browser
